# About
欢迎来到我的每日论文摘要项目！这里汇总了通过 AI 自动生成的论文报告。
This tool will daily crawl https://arxiv.org and use LLMs to summarize them.

% See in: https://dw-dengwei.github.io/daily-arXiv-ai-enhanced/

# How to use
This repo will daily crawl arXiv papers about **math.NT and math.RT**, and use **Gemini** to summarize the papers in **Chinese**.
If you wish to crawl other arXiv categories, use other LLMs or other languages, please follow the instructions in https://github.com/dw-dengwei/daily-arXiv-ai-enhanced/ and in https://github.com/dan211505017/daily-arXiv-gemini-enhanced/.
